Exports

- `default Icons(options)` — Astro integration preset that wires Starlight + this plugin
- `starlightIconsPlugin(options)` — Starlight plugin (sidebar, codeblock CSS, Expressive Code)
- `starlightIconsIntegration(options)` — Astro integration (safelist generator)
- `withSidebarIcons(sidebar)` — helper to transform `icon` → `attrs.data-icon`
- `pluginIcon()` — Expressive Code plugin used internally
