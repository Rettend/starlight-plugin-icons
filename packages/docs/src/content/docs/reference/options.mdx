`StarlightIconsOptions`

- `sidebar` boolean — override sidebar component (default: `true`)
- `extractSafelist` boolean — scan content and write safelist (default: `true`)
- `codeblock` boolean — enable codeblock CSS and hook (default: `true`)

Preset `Icons({ starlight, ...icons })`

- `starlight` — forwarded to `@astrojs/starlight`, with optional `sidebar` supporting `icon` helper field
