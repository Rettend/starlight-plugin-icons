---
title: Custom Icons
description: Add your own Iconify collections and tweak appearance.
---

Add a collection via UnoCSS

```ts title="uno.config.ts"
import fs from 'node:fs/promises'
import { defineConfig, presetIcons } from 'unocss'
import { presetStarlightIcons } from 'starlight-plugin-icons/uno'

export default defineConfig({
  presets: [
    presetStarlightIcons(),
    presetIcons({
      collections: {
        icons: {
          drizzle: () => fs.readFile('./src/assets/drizzle.svg', 'utf-8'),
        },
      },
      extraProperties: { display: 'inline-block', 'vertical-align': 'middle' },
      customizations: {
        iconCustomizer(collection, icon, props) {
          if (['devicon', 'simple-icons', 'logos'].includes(collection)) props.transform = 'scale(0.8)'
          if (collection === 'icons' && icon === 'drizzle') props.transform = 'scale(0.8)'
        },
      },
    }),
  ],
})
```

Use it

Example source

````mdx
<IconLink href="/demo/drizzle/" icon="i-icons:drizzle" title="Drizzle" />
````

Rendered

import IconLink from 'starlight-plugin-icons/components/IconLink.astro'

<div class="row">
  <IconLink href="#" icon="i-icons:drizzle" title="Drizzle" />
  <IconLink href="#" icon="i-ph:github-logo-duotone" title="GitHub (ph)" />
</div>

Tips

- Namespace your collection to avoid clashes (e.g. `icons:*`).
- Use `customizations.iconCustomizer` to scale or style specific sets.
