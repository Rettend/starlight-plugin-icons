<Steps>

1. ##### Enable the sidebar feature

    ```js title="astro.config.mjs" ins={2}
    Icons({
      sidebar: true,
      starlight: { ... },
    })
    ```

2. ##### Configure UnoCSS to discover the icons

    Set `extractSafelist: true` to automatically generate a safelist,
    or if you prefer not to do that, you need to configure UnoCSS to discover the icons.

    <Tabs syncKey="safelist-strategy">
      <TabItem label="Automatic safelist">

      Enable extraction and add the cache to `.gitignore`.

      ```js title="astro.config.mjs" ins={3}
      Icons({
        sidebar: true,
        extractSafelist: true,
        starlight: { ... },
      })
      ```

      ```txt title=".gitignore"
      .starlight-icons
      ```

      </TabItem>
      <TabItem label="Configure UnoCSS">

      Point UnoCSS to your `astro.config.mjs` so icon classes in the sidebar are extracted.

      ```ts title="uno.config.ts" ins={2-11} "|mjs" "'astro.config.mjs'"
      export default defineConfig({
        content: {
          pipeline: {
            include: [
              /\.(vue|svelte|[jt]sx|vine\.ts|mdx?|astro|elm|php|phtml|html|mjs)($|\?)/,
            ],
          },
          filesystem: [
            'astro.config.mjs',
          ],
        },
      })
      ```

      </TabItem>
    </Tabs>

3. ##### Install an Icon collection

    Choose any collection. See the <Link href="https://icones.js.org/">Icon Gallery</Link>.

If you are using `starlight-sidebar-topics`, see [configuration](/guides/configuration/#using-with-starlight-sidebar-topics) on how to integrate the two plugins.

## Example

Add an `icon` field to each sidebar entry.

Format: `i-<collection>:<name>`

You can also add label-only entries (no `slug`/`link`) when you want a non-clickable row in the sidebar.

<br />

```js title="astro.config.mjs" "icon: 'i-ph:rocket-launch-duotone'"
Icons({
  starlight: {
    sidebar: [
      {
        label: 'Guides',
        items: [
          { icon: 'i-ph:rocket-launch-duotone', label: 'Intro', slug: 'guides/intro' },
        ],
      },
    ],
  },
})
```

## Customization

Set CSS variables to configure sidebar icon size and spacing.

```css title="src/styles/custom.css"
:root {
  --spi-sidebar-icon-size: 1.25rem;
  --spi-sidebar-icon-gap: 0.25rem;
}
```
