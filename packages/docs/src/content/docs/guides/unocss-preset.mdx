---
title: UnoCSS Preset
description: Use the preset to include internal icons and safelist.
---

Add the preset

```ts title="uno.config.ts"
import { defineConfig } from 'unocss'
import { presetStarlightIcons } from 'starlight-plugin-icons/uno'

export default defineConfig({
  presets: [presetStarlightIcons()],
})
```

What it does

- Registers a small internal icon collection (folders, svelte-js/ts).
- Loads a generated Material Icons safelist so Purge does not strip used icons.
