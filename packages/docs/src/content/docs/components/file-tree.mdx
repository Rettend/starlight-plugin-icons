import FileTree from 'starlight-plugin-icons/components/FileTree.astro'

Example source

````mdx
import FileTree from 'starlight-plugin-icons/components/FileTree.astro'

<FileTree>
  - src/
    - pages/
      - index.astro
  - package.json an **important** file <span class="i-ph:warning-duotone"></span>
</FileTree>
````

Rendered

<FileTree>
  - docs/
    - components/
      - file-tree.mdx
  - **uno.config.ts** the <span class="i-logos:unocss"></span> UnoCSS file!
  - package.json an **important** file <span class="i-ph:warning-duotone"></span>
</FileTree>

Customization (scoped)

<div class="demo-filetree-custom">
  <FileTree>
    - src/
      - routes/
        - index.ts
    - README.md
  </FileTree>
</div>

<style>{`
  .demo-filetree-custom {
    --spi-filetree-icon-size: 1.2rem;
    --spi-filetree-icon-gap: 0.5rem;
  }
`}</style>

Authoring rules

- Directories end with `/` or contain nested lists.
- `**README.md**` highlights an entry.
- Text after the name becomes a comment: ``README.md important``.

Accessibility

- Directories include an SR-only label for screen readers.

CSS variables

```css
/* File tree */
--spi-filetree-icon-size: 1.1rem;
--spi-filetree-icon-gap: 0.25rem;
```
